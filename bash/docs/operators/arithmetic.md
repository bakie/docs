### Addition
```bash
#!/usr/bin/env bash
a=2
b=3
c=$((a + b))
echo ${c}

Output: 5
```

### Subtraction
```bash
#!/usr/bin/env bash
a=2
b=3
c=$((a - b))
echo ${c}

Output: -1
```

### Multiplication
